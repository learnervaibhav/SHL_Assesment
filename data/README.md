# Data Directory

This directory stores generated data files for the SHL Assessment Recommender.

## Files

- **catalog.json** - Complete SHL assessment catalog (generated by `scripts/fetch_catalog.py`)
  - Contains normalized assessment objects with fields: id, name, url, description, job_levels, keys, languages, duration
  - Auto-generated; do not edit manually

- **assessments.json** - Processed assessment index with additional metadata (generated by Phase 1.2)
  - Includes inverted indexes and keyword extraction
  - Auto-generated during indexing phase

- **embeddings.json** - Semantic embeddings for assessments (generated by Phase 1.3)
  - Vector representations using Sentence-Transformers (all-MiniLM-L6-v2)
  - Auto-generated; used for semantic search

## Generation

To regenerate all data files:

```bash
# 1. Fetch and normalize catalog
python scripts/fetch_catalog.py

# 2. Build assessment index and generate embeddings
python scripts/index_assessments.py
```
